\doxysubsection{Detailed Description}
Declarations of functions related to simulation and grid operations. 

This header file contains declarations for functions used in simulating a partial differential equation using finite difference methods. It also declares functions related to initializing the grid, applying boundary conditions, and updating the Laplacian.
